# Tarea 1: Lista Doblemente Enlazada en Python

## Datos del Estudiante
- **Nombre:** Nombre Completo
- **Carnet:** Número de Carnet

## Descripción de tarea 1
en esta tarea voy a implementar estructura de datos de Lista Doblemente Enlazada que almacena información de estudiantes (Nombre, Apellido, Carnet) y genera visualizaciones automáticas utilizando Graphviz.

## Requisitos
- Python 3.x
- Se debe instalar Graphviz en la computador.
- Librería de Python `graphviz`.

## Instrucciones de Uso
1. Ejecutar el archivo `main.py`.
2. Utilizar el menú interactivo para insertar o eliminar nodos.
3. Al realizar cualquier operación, se actualizará el archivo `lista_doble_visual.png` con la representación gráfica de la lista.
